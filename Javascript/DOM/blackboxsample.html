Number

A number is a numeric value.

Example: let num = 31;

String

A string is a sequence of characters.

Example: let name = "Divya";

Boolean

A boolean is a logical value that can be either true or false.

Example: let isBoolean = true;

Array

An array is a collection of values that can be of any type.

Example: let fruits = ["apple", "banana", "orange"];

Null

Null is a special value that represents the absence of a value.

Example: let nothing = null;

Undefined

Undefined is a special value that represents a variable that has not been assigned a value.

Example: let x;

Object

An object is a collection of key-value pairs.

Example: let person = { name: "Divya", age: 20, isStudent: true };
